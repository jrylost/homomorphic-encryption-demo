# homomorphic-encryption-demo
 Python demo of homomorphic encryption scheme BFV

这个仓库是我们对全同态加密方案[BFV](https://eprint.iacr.org/2012/144.pdf)的开源实现：
* 加解密方案
* 加法乘法
* Relinearization(Version 1)
